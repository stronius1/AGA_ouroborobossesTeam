# ADR-0025: Сервис обогащения кредитных заявок

## Контекст

Кредитному конвейеру нужно обогащать заявки внешними атрибутами.

## Решение

Вызывать сервис AS-0099 по версионируемому API-контракту.

## Альтернативы

Рассматривались локальный справочник и ручная обработка.

## Последствия

Потребуются SLA, мониторинг и режим деградации при недоступности сервиса.

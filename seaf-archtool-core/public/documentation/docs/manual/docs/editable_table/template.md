# Описание

Плагин для отображения табличных данных, их редактирования и сохранения

# ⚠️ Важные особенности ⚠️

В документации к SEAF.ArchTool редактирование отключено.
Вы можете включить редактирование вручную или воспользоваться примером в [данном репозитории](https://gitverse.ru/seafteam/seaf-examples).

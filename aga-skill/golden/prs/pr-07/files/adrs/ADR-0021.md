# ADR-0021: Вывод легаси-скоринга из кредитного контура

## Контекст

AS-0010 имеет статус `eliminate`. Его оценка уже воспроизведена на AS-0012,
поэтому поток IF-0068 от кредитного конвейера создаёт лишнюю зависимость.

## Решение

Удалить IF-0068 и исключить AS-0010 из целевой диаграммы. Кредитный конвейер
использует утверждённый API AS-0012.

## Альтернативы

Рассматривались параллельная эксплуатация и прокси перед AS-0010. Оба варианта
сохраняют зависимость от выводимой системы и отклонены.

## Последствия

После сверки результатов AS-0010 может быть остановлена, а её сопровождение —
прекращено по плану декомиссии.

# ADR-0007: Переход витрины отчётности на колоночное хранилище

## Контекст
Текущие витрины на строковом хранилище не выдерживают ad-hoc нагрузку
аналитиков: p95 тяжёлых запросов превышает 40 секунд.

## Решение
Перевести витрину AS-0007 на колоночное хранилище с материализацией
витринных слоёв и инкрементальной загрузкой.

## Последствия
Потребуется миграция ETL-конвейеров и переобучение команды аналитики.

deterministic review, MCP prepare/finalize contract, frozen evaluation basket и offline
demo. Не проверены pinned upstream trees, ArchTool UI, official-agent loop,
clean clone, public repository и video. Поэтому Project Results readiness не заявляется
как `done`.

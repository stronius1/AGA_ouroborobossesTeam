# Добро пожаловать в SEAF.ArchTool!👋


**SEAF.ArchTool** - инструмент управления архитектурой как кодом.

**SEAF (Sber Enterprise Architect Framework)** представляет собой набор методов и инструментов, позволяющих управлять корпоративной архитектурой Группы Сбер. Он подойдет и другим компаниям, потому что является open sourse и его можно адаптировать под разные нужды.

Документация на SEAF.ArchTool доступна в [репозитории GitVerse](https://gitverse.ru/seafteam/seaf-archtool-core).

Присоединяйся к [сообществу в Телеграм](https://t.me/+VyrIlFp_KSbrD3-c), где тебе всегда помогут!

# Быстрый старт 🚀

Чтобы заменить эту стартовую страницу, создай собственный документ с идентификатором
dochub.welcome по примеру:

```yaml
  ...
  docs:
    dochub.welcome:
      title: Моя стартовая страница
      type: markdown      # Тип документа
      source: welcome.md  # Файл с документом
    ...
```

Подробнее о создании документов и их видах [тут](https://gitverse.ru/seafteam/seaf-archtool-core/content/master/public/documentation/docs/manual/docs/docs.md?blank).
